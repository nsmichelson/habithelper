# Comprehensive Tags Reference Guide - Habit Helper Tips Data

This reference lists the canonical tag values in the Habit Helper tip library after the audit that replaced broad categories with specific intent-driven tags. The guide focuses on the five active habit areas (Exercise, Nutrition, Organization, Productivity, and Relationships) and surfaces the curated goal, helps-with, feature, effort, and cost tags available for filtering and UI copy.

---

## Global Conventions

- **Time buckets** are now standardized to **0-5**, **5-15**, **15-30**, **30-60**, and **60+ minutes**. Legacy buckets (e.g., `planning`, `ongoing`, `30-45`) should be mapped into these ranges when rendering filters or analytics.
- **Effort** values are expressed as the qualitative rollups `minimal`, `low`, `medium`, and `moderate`, derived from combined mental/physical load scores.
- **Cost** uses three tiers: `$` (no/very low cost), `$$` (low recurring cost or one-off purchase), and `$$$` (premium or ongoing expense). Current tips only use `$` and `$$`.
- Feature tags are additive. Multiple feature tags may appear on a single tip to communicate accessibility traits.

## Usage Notes

- Tags are additive—tips frequently contain multiple goals, helps-with, and features to support layered filtering.
- When building UI filters, display time buckets in ascending order and group costs from `$` → `$$` → `$$$` even if the higher tiers have zero matches yet.
- Audit-driven tags emphasize observable behaviors and user-facing language; avoid reintroducing legacy umbrella tags (`better_habits`, `build_healthy_habits`, etc.).
